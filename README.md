# RBM-Plankton

Restricted Boltzmann Machine for unsupervised learning of plankton community 
structure from Lake Greifen monitoring data (2019–2024).

## Motivation

Lake Greifen exhibits seasonal plankton dynamics. This project uses RBM to learn latent community states from 83 taxa abundance time series and interprets learned patterns against known ecological cycles.

**Data**: Eyring et al. (2025). *Scientific Data* 12:653 — 5 years of high-frequency 
phytoplankton and zooplankton observations.

## Setup

### Using uv (recommended)

```bash
# One-time install
sudo apt install uv

# Setup environment
uv venv
source .venv/bin/activate
uv sync
```

### Using pip

```bash
sudo apt install python3-venv
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage

Each script handles one phase. Run in order or independently:

```bash
# 1. Clean and prepare data (drop NaN, log-transform, z-score, split)
python src/preprocess.py

# 2. Train RBM
python src/train.py

# 3. Interpret: hidden activations, weight matrix, seasonal patterns
python src/analyse.py
```

See script docstrings for parameters and output locations.
